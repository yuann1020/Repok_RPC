# Pickleball Court Booking Website

## Overview
A premium, full-stack web application designed for a single pickleball venue. Features include discovering available courts, booking slots securely, and making online payments. It includes a complete Admin dashboard to govern courts, availability, and bookings. 

Built with:
- **Frontend**: Next.js 14 (App Router) + Tailwind CSS
- **Backend**: NestJS + PostgreSQL
- **ORM**: Prisma
